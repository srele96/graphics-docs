# Move vertex on the plane

Desmos demo:

<https://www.desmos.com/3d/jd6xywrfks>

The math and visual demo of moving vertex along the plane.

Essentially, any function capable of outputting a shape, a cube, a plane, a mountain, etc, is capable of outputting a vertex following a trajectory on itself. The function is a coordinate mapping. It can create a set of vertices for a shape. It can also create a vertex, and through manipulation of the input, we can control it's trajectory.
